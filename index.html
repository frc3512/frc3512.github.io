Who We Are

We are the software subteam of FRC Team 3512 located in Orcutt, California. We teach high school students practical software and controls engineering skills through competitive robotics.


What We Strive For

Produce Software engineers That:

  • Understand the tradeoff between maintainability and performance.
  • Can recognize when a given programming language or feature is the right fit for a problem.

That students learn:

  • The software subteam programs the robot to make it move and automate parts of its functionality.
  • All the tools they need to succeed.

What You Will Learn With Us

  • How to think like a programmer
  • What tools programming languages offer, and when/how to apply them solve problems in a manner that is effective, efficient, and promotes the long-term maintainability of the software project.
  • The tools and methodologies software engineers use to collaborate with each other and manage the complexity of a software project.
  • Software engineering principles for real-time and embedded systems.
  • Control theory, which lets us do two things:
    • Model our robots mathematically so we can test our software's correctness before the hardware is completed.
    • Design and test controllers for those models that ensure actions like "move the elevator to this position" will work with minimal trial-and-error on the real hardware.
  • How to write automated tests that ensure our software continues to work as we add features and perform cleanups.

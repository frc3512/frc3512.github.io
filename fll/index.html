Set up Wi-Fi based cameras

Find the IP

  • Try a previous IP
  • Use the amcrest tool to find it
  • Use ping

If the camera cannot be found, it may not be connected to Wi-Fi. Use the amcrest app to connect it. Refer to the data portion for more information about the cameras.

Open the stream in VLC

Open VLC and go to "Open Stream" in the task bar. Use rtsp://username:password@IP-Address to connect to and start the stream.

Ensure that the cameras are oriented correctly and video quality settings are okay.

In the event there are video quality or bandwidth issues, feel free to edit the camera settings if you know how and are confident in what you're doing.
